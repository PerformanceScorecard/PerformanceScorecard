Performane Scorecard

Thanks for installing Performane Scorecard

This extension is a community supported replacement for the Google PageSpeed Insights extension. The original Google extenion is no longer available in the Chrome store, but this community supported version uses the same rules engine and offers the same functionality.

Performance Scorecard is executed from the Chrome Developer Tools. To use Performance Scorecard:

  1. Open Chrome Developer Tools by selecting the Chrome menu at the top-right of your browser window, then select Tools -> Developer tools.
  2. In the Chrome Developer Tools pane, click the Performance Scorecard tab to switch to the Performance Scorecard panel.
  3. Navigate Chrome to a web page, and click the red ANALYZE button to analyze the page.
How to hide the Performance Scorecard button

Once you've found Performance Scorecard in the Chrome Developer Tools, you may wish to remove the Performance Scorecards icon from the main Google Chrome toolbar. To hide the Performance Scorecards icon, right click on it and select "Hide Button".
